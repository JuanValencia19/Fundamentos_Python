# 🟢 PARTE 1 — CONDICIONALES (8 ejercicios)

## 1️⃣ Número positivo, negativo o cero
Pide un número al usuario y determina si es:
- Positivo
- Negativo
- Cero

---

## 2️⃣ Número par o impar
Pide un número entero y determina si es par o impar.

---

## 3️⃣ Mayor de edad
Pide la edad del usuario y muestra:
- "Mayor de edad" si tiene 18 o más
- "Menor de edad" si tiene menos de 18

---

## 4️⃣ Comparación de dos números
Pide dos números y muestra:
- Cuál es mayor
- O si son iguales

---

## 5️⃣ Clasificación de nota
Pide una nota del 0 al 100 y clasifica:
- 90–100 → Excelente
- 70–89 → Bueno
- 50–69 → Regular
- Menos de 50 → Reprobado

---

## 6️⃣ Apuesta ganadora
Pide:
- Cuota decimal
- Resultado (1 si ganó, 0 si perdió)

Si ganó:
    calcula ganancia = 100 * cuota
Si perdió:
    muestra "Apuesta perdida"

---

## 7️⃣ Validación de contraseña
Define una contraseña guardada en una variable.
Pide al usuario que la ingrese.

Si coincide → "Acceso concedido"
Si no → "Acceso denegado"

---

## 8️⃣ Clasificación de probabilidad implícita
Pide una cuota decimal.
Calcula probabilidad = 1 / cuota.

Si probabilidad > 0.6 → Favorito claro
Si entre 0.4 y 0.6 → Partido equilibrado
Si menor a 0.4 → No favorito
