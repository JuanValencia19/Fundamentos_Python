# 🐍 Python Fundamentals Journey

Este repositorio está dedicado exclusivamente al estudio y práctica de los fundamentos de Python.

## 🎯 Objetivo

Construir una base sólida en Python antes de avanzar hacia:

- Análisis de datos
- Estadística aplicada
- Modelos predictivos
- Machine Learning

La meta principal es dominar la lógica de programación y la sintaxis del lenguaje desde cero.

---

## 📚 Contenido

En este repositorio se practican:

- Variables
- Tipos de datos
- Operaciones matemáticas
- Entrada y salida de datos
- Condicionales
- Bucles
- Estructuras de datos (listas, diccionarios, etc.)
- Funciones
- Manejo básico de errores

Cada carpeta representa módulo de estudio y la solución individual de los programadores Juanjo y Santiago.
